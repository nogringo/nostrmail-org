# Nostr Mail

The official landing page for the **Nostr Mail Protocol** and the **Nmail** reference client.

Nostr Mail is an open standard designed to replace traditional SMTP with a decentralized, encrypted, and sovereign relay network.

## 🚀 Overview

- **Protocol:** Defines Kind 1301 for email transport, NIP-44 for encryption, and NIP-59 (Gift Wraps) for metadata privacy.
- **Client:** Nmail is a cross-platform (Web, Linux, Android) reference implementation built with Flutter.
- **Sovereignty:** Your identity is your `npub`. No central authority can block your address or read your mail.

## 🛠️ Built With

- **Astro:** High-performance static site generator.
- **Vanilla CSS:** Optimized for speed and accessibility.
- **Adaptive Design:** Automatic light/dark mode based on system preferences.
